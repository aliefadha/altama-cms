## Project Overview

Altama CMS is a content management system designed to manage blog posts, product information, and media galleries for an external landing page. The CMS provides a web-based interface for content creators to manage content that will be consumed by an external landing page through REST API endpoints.

### Key Technologies

- **Frontend**: React 19, TypeScript, Vite
- **Routing**: TanStack Router (file-based routing)
- **Data Fetching**: TanStack Query
- **UI Components**: Shadcn UI with Tailwind CSS
- **Authentication**: Simple authentication system
- **Integration**: REST API endpoints for external landing page consumption

### Core Features

- Blog post management (create, edit, delete, publish)
- Product information management
- Media gallery management
- Simple authentication system
- REST API endpoints for external consumption
- Responsive design with modern UI components
